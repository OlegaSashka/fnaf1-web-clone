# FNAF 1 Clone (веб-версия)

Клон Five Nights at Freddy's 1 на чистом HTML + Canvas + JavaScript.

## Технологии

- HTML5 Canvas 2D
- requestAnimationFrame
- Anime.js (анимации)
- Howler.js (звук)
- Только vanilla JS + лёгкие библиотеки через CDN

## Запуск

1. Склонировать репозиторий
2. Открыть `start.bat`
